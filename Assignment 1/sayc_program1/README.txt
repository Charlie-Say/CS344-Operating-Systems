1. To compile the code

    - Locate directory of code files

    - Unzip files

    - Locate directory of unzipped files

    - At the command line type in "gcc -o movies main.c"

    - This will create a .exe executable file


2. Running the program

    - At the command line type in "./movies.exe your_file_name_directory"
            example: "./movies.exe movies_sample_1.csv"

    - This will execute the program with the file name as the argument
